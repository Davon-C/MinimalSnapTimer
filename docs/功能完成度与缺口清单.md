# 功能完成度与缺口清单

## 已完成
- 普通倒计时、暂停、继续、停止、重置
- 秒表模式
- 坐 40 / 站 15 工作流与预设动态按钮
- 纯时间模式
- 点击穿透与托盘恢复入口
- 多显示器窗口保存 / 恢复 / 安全回退
- 托盘菜单、托盘图标、托盘状态显示
- 系统通知与托盘气泡降级
- 浅色 / 深色主题
- 简体中文 / English 基础切换
- 启动诊断日志、坏配置回退、启动保护
- 单文件 self-contained 完整版发布

## 部分完成
- 跟随系统主题
  现状：支持跟随系统判断，但不保证实时监听系统变化。
- English 文案覆盖
  现状：核心路径已覆盖，边角文本仍可能存在少量遗漏。
- 原生 Toast
  现状：已支持基础系统通知与失败降级，但不是复杂交互版 Toast。

## 本轮重点收口
- 主窗口 UI 重排
- 设置窗口 UI 重排
- 纯时间模式提醒弹窗 Owner 异常修复
- 配置文件 `settingsVersion` 迁移
- 发布包收口为单一完整版

## 当前已知限制
1. 跟随系统主题不一定实时响应系统主题切换。
2. 英文下少量边角文本可能未完全覆盖。
3. 原生 Toast 仍是轻量实现。
4. 当前不提供 lite 包，只保留完整版。

## 后续建议
1. 如需继续打磨体验，优先补系统主题实时监听。
2. 再下一步可补托盘状态图标细分，而不是重做功能架构。
