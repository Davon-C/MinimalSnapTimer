# 第二轮增强计划

## P0：影响基本使用或明显界面问题

1. 补齐纯时间模式点击穿透的直接关闭入口  
   当前点击穿透可以通过设置开关控制，但缺少更直接的托盘快捷关闭入口。

2. 补齐提醒声音停止动作  
   自定义提醒音可以循环播放，但提醒弹窗尚未把“停止循环声音”接到 `NotificationService.StopLoopingSound()`。

3. 实现原生 Windows Toast 通知  
   当前只有提醒弹窗和托盘气泡，没有真正的原生 Windows Toast。

4. 统一第二轮界面细节验收  
   重点检查托盘菜单、纯时间模式菜单、任务栏进度条、标题前缀和点击穿透交互，避免出现功能有代码但入口不顺手的问题。

## P1：影响体验但不阻塞使用

1. 补齐深色主题和跟随系统主题
2. 完整补齐预设编辑字段  
   包括完成动作、自动切换下一个预设、下一个预设目标等。
3. 完善全局快捷键体验  
   包括注册失败提示、自定义编辑和冲突处理。
4. 优化本地日志  
   将设置开关真正接到日志输出，并补更多关键错误点。
5. 完善任务栏与托盘状态表达  
   包括更细的图标状态、标题前缀和托盘提示说明。

## P2：增强功能

1. 增加运行中滴答声
2. 增加颜色选择器和更直观的透明度设置
3. 增加更完整的提醒方式组合
4. 增加多语言框架  
   先做中文和英文双语，再决定是否扩展。
5. 增加更完整的预设管理体验  
   包括导入导出、批量操作、更多预设类型说明。

## P3：可选功能

1. 更丰富的纯时间模式视觉效果
2. 自定义托盘图标资源
3. 更完整的首次启动引导
4. 文档截图、演示图和操作录屏
5. 可选安装器或绿色版分发说明优化

## 建议执行顺序

1. 先做 P0，保证提醒、点击穿透和通知这类高频功能顺手可用。
2. 再做 P1，补齐主题、预设编辑和快捷键这些明显体验缺口。
3. 最后按需要推进 P2 和 P3，避免在基础体验尚未打磨完成前分散精力。
